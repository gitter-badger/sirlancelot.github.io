The Quine Website

I first heard of quines long ago only after I'd been programming for a while. The concept intrigued me. It wasn't until I saw someone else apply it to a website that I decided that's what I was going to do for my own site.

Strictly speaking, this is only a spiritual quine. In a true quine the entire program outputs itself. For the sake of readability of this website, I'm using a few external resources which can be found in the actual source of this site.
